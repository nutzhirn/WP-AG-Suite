# WP-AG Roadmap

Diese Datei fasst die zentrale Roadmap der WP-AG Plattform zusammen. Detaillierte Einzelroadmaps befinden sich in den Dokumentationsordnern.

## WP-AG Suite

WP-AG Suite ist die WordPress-Werkzeugsammlung für Diagnose, Migration, Medienverwaltung, Dokumentenverwaltung, Sicherheit, Performance und Qualitätskontrolle.

### Bestand

- Dashboard
- Diagnose
- Security Check
- Beitragsimport
- BookViewer
- MediaInspector
- BackupManager
- Contacts
- PostTools
- RevisionManager
- FontForce
- MenuSearch
- KatCloud
- Artikelarchiv
- ThemeCleaner
- Object Cache Analyse

### Geplante Erweiterungen

- Seitenimport
- Kommentarimport
- Erweiterte Migration
- Benutzer-Mapping mit Anlage fehlender Benutzer
- PDF-zu-BookViewer-Konvertierung
- Avatar Modul
- Image Handling
- Akkordeon Generator
- Image Resizer
- Child Theme Generator
- Activity Log / Audit Log
- Adressverwaltung
- Socializer
- Performance & Diagnostics Center
- Website Quality Report

## WP-AG Trial

WP-AG Trial soll eine vollständige Plattform für Border-Collie-Trials und Hütehundveranstaltungen werden.

Schwerpunkte:

- Eventmanagement
- Handler- und Hundeverwaltung
- Nennungen und Anmeldung
- Startlisten
- Zeitplanung
- Richter- und Laufverwaltung
- Bewertungskriterien
- OLF-Regel bei Punktegleichheit
- Flexible Wettbewerbstypen
- Ergebnisse, Ranglisten und Statistik
- Frontend-Ausgabe
- Mehrsprachigkeit
- Rollen und Berechtigungen

## WP-AG Verein

WP-AG Verein soll eine datenschutzfreundliche Vereins- und NGO-Verwaltung werden.

Schwerpunkte:

- Mitgliederverwaltung
- Mitgliedschaftsarten
- Beitragsverwaltung
- Kommunikation
- Veranstaltungen
- Dokumente
- Vorstand und Ausschüsse
- Mitgliederportal
- DSGVO- und DSG-Unterstützung
- Lokales Hosting sensibler Daten
- Rollen und Berechtigungen
- Activity Log

## WP-AG Developer

WP-AG Developer ist eine spezialisierte Edition für Agenturen, Entwickler und Administratoren.

Schwerpunkte:

- Code Inspector
- PHP Analyse
- WordPress Coding Standards
- Sicherheitsprüfung von Plugins
- Deprecated Functions Scanner
- Hook Explorer
- Database Tools
- REST API Explorer
- Cron Manager
- Plugin Generator
- Child Theme Generator
- Debug Center
- WP-CLI Integration
