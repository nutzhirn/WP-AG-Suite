# WP-AG-Suite
WP-AG ist als schlanke Produktfamilie gedacht: klar getrennte Module, nachvollziehbare Funktionen und ein humanistischer Anspruch an digitale Werkzeuge - der Mensch ist im Mittelpunkt. Die Software soll Organisationen, Vereinen und engagierten Menschen Arbeit abnehmen, ohne sie in technische Abhängigkeiten zu drängen.
