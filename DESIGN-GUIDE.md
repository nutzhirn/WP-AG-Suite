# WP-AG Design Guide

## Grundprinzipien
- Dashboard = Übersicht
- Module = Arbeiten
- Kachel = Aktion
- Keine Informationsdopplungen
- Inhalte linksbündig

## Raster
- 6er Raster für Status
- 4er Raster für Hauptfunktionen
- 3er Raster für Aktionen
- 2er Raster für große Werkzeuge

## Gestaltung
- Gleiche Höhen innerhalb einer Reihe
- Gleiche Kacheltypen = gleiche Höhe
- Tabellen volle Breite
- Einheitliche Farben
- Einheitliche Abstände

## Sprache
- Verständliche Begriffe
- Sicherheitseinstellungen statt Härtung
- Ursachen statt nur Bewertungen
