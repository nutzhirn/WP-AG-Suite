# MediaInspector

## Zweck

Dieses Modul ist Teil der WP-AG Suite und erfüllt eine klar abgegrenzte Verwaltungs- oder Analyseaufgabe innerhalb von WordPress.

## Entwicklungsziel

Das Modul soll sich in die gemeinsame Oberfläche, das gemeinsame Logbuch, einheitliche Buttons, einheitliche Tabellen und konsistente Import- und Exportmuster der Suite einfügen.

## Qualitätsanforderungen

- WordPress-konforme Umsetzung.
- Übersetzbare sichtbare Strings.
- Sanitizing von Eingaben.
- Escaping von Ausgaben.
- Capability Checks.
- Dokumentierte Bedienung.
- Keine unnötige Abhängigkeit von externen Diensten.
