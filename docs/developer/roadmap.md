# WP-AG Developer – Kanonische Roadmap

WP-AG Developer ist eine spezialisierte Edition für Agenturen, Entwickler und Administratoren.

## Code Inspector

Analyse von Plugin- und Theme-Code.

## PHP Analyse

Prüfung von PHP-Dateien, Syntax, Kompatibilität und typischen Fehlerquellen.

## WordPress Coding Standards

Unterstützung bei der Einhaltung der WordPress Coding Standards.

## Sicherheitsprüfung von Plugins

Prüfung auf typische Sicherheitsprobleme wie unsanitized Input, unescaped Output, fehlende Nonces und fehlende Capability Checks.

## Deprecated Functions Scanner

Erkennung veralteter WordPress- und PHP-Funktionen.

## Hook Explorer

Anzeige und Suche von Actions und Filtern.

Funktionen:

- Actions anzeigen.
- Filter anzeigen.
- Hook-Dokumentation.
- Hook-Suche.

## Database Tools

Datenbankwerkzeuge für:

- Tabellenanalyse.
- Datenbankoptimierung.
- Query Analyse.

## REST API Explorer

Werkzeug zur Prüfung von REST-Endpunkten.

Funktionen:

- Endpunkte testen.
- Authentifizierung prüfen.
- JSON-Vorschau.

## Cron Manager

Verwaltung von WordPress-Cronjobs.

Funktionen:

- Cronjobs anzeigen.
- Ausführungen protokollieren.
- Cronjobs manuell starten.
- Fehler analysieren.

## Plugin Generator

Erzeugung eines Grundgerüsts für neue WordPress-Plugins unter Berücksichtigung von WordPress-Standards.

## Child Theme Generator

Erzeugung von Child Themes. Dieses Modul kann sowohl in der Suite als auch in Developer sinnvoll sein, passt aber besonders gut in die Developer Edition.

## Debug Center

Werkzeuge für:

- Debug Log Viewer.
- Error Tracking.
- Performance Messungen.

## WP-CLI Integration

Integration von WP-CLI.

Funktionen:

- Befehle ausführen.
- Dokumentation anzeigen.
- Ergebnisse protokollieren.
